Isolated step scope

This step has an isolated step scope, which means we cannot manipulate our data directly. Instead, we need to grab our data when a step is instanciated (i.e. when our step controller is instanciated), and we need to save our changes when we navigate away.

In an isolated step controller, we can inject multiStepFormScope which is our directive's scope. And we can use the $destroy event to save any data which has been modified by the user.
