Using forms

You can use forms in steps, however this doesn't change much. Each step can take a property hasForm (true or false, false by default) but this property is harmless: it doesn't do anything at all.

This module is not opiniated about how to use forms. Have you ever used Angular UI bootstrap modals? Steps are very much like modals: isolated units of business logic. Therefore, validation and decision to whether allow navigation to the next step should be performed in each step.
