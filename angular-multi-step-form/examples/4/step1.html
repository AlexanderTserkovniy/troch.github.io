Markup structure and class names

Creating step transitions is made easy by the markup structure and class names added by this module.

Animations are performed with $animate. You need to add ngAnimate as an application dependency:

  • A leaving step .form-step will have .ng-leave and .ng-leave-active classes added to its main element if a CSS transition is detected.
  • An entering step will have .ng-enter and .ng-enter-active classes added to its main element if a CSS transition is detected.
